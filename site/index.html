Deploy Your Way

Bare metal, Docker, or fleet management.

⚡ Bare Metal

Full installation with systemd, nginx dashboard, and optional Ollama.

curl -sO https://raw.githubusercontent.com/ivanuser/cortex-server-os/main/install.sh && sudo bash install.sh

🐳 Docker

Quick containerized deployment with optional Ollama sidecar.

git clone https://github.com/ivanuser/cortex-server-os.git && cd cortex-server-os && docker compose up -d

🏗️ Fleet

Multi-node management with central dashboard and health monitoring.

git clone https://github.com/ivanuser/cortex-management-server.git && cd cortex-management-server && npm start
